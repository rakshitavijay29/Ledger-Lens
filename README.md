# About the Project

**Ledger Lens** is an end-to-end data analytics and machine learning project developed to identify suspicious banking transactions through anomaly detection.

The project performs the complete machine learning workflow—from data preprocessing and exploratory data analysis to feature engineering and fraud detection using the **Isolation Forest** algorithm.

The objective is to assist financial institutions in identifying unusual transaction patterns that may indicate fraudulent activities.

---

# Problem Statement

Financial institutions process millions of transactions daily. Manual fraud detection is time-consuming and often ineffective for identifying hidden anomalies.

Ledger Lens addresses this challenge by applying machine learning techniques to automatically detect potentially suspicious transactions based on transaction behavior.

# Machine Learning Approach

The project utilizes the **Isolation Forest Algorithm**, an unsupervised anomaly detection model specifically designed to identify rare observations in large datasets.

### Why Isolation Forest?

- Efficient on large datasets
- Does not require labeled data
- Detects rare transaction patterns
- Low computational complexity
- Widely used in fraud detection systems
